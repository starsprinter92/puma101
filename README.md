# Puma101
A mostly faithful Mac OS X Cheetah/Puma recreation, all on XFCE.

## Features

- Aqua GTK and XFWM themes
- Plank Reloaded Dock
- Accurate fonts
- HD Aqua wallpapers

## Requirements

- XFCE 4
- GNU Stow
- Plank Reloaded
- `xfce4-appmenu-plugin-git`
- GTK 2/3 support
- `ttf-lucida-fonts`
- `pcmanfm`
- `gvfs`

## Install
``stow -t ~ gtk icons plank xfce4``

Also you need to make plank run on startup
and `pcmanfm --desktop`
